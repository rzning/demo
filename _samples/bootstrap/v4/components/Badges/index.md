## Badges 徽章

```html
<span class="badge badge-default">徽章</span>
```

### 语义颜色

Class | Color | Description
--|--|--
`badge-default` | 灰色 | 默认
`badge-primary` | 蓝色 | 主要
`badge-success` | 绿色 | 成功
`badge-info` | 青色 | 信息
`badge-warning` | 黄色 | 警告
`badge-danger` | 红色 | 错误

### 药丸徽章

使用药丸样式添加 `badge-pill` 类即可。
